
### طريقة تشغيل الواجهة
1) نسخ المشروع الى جهازك الشخصي
2) التوجه داخل المشروع
3) كتابة الامر المعطى لتشغيل تطبيق  `Angular`
``` shell
#1
git clone git@github.com:fahad5359/ChatBot-Interface.git 

#2
cd ChatBot-Interface

#3
ng serve
```
